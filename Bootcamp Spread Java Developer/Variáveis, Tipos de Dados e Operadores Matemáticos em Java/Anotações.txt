Variáveis: Um espaço na memória do computador, onde se pode guardar valores.

Quatro tipos: instância: Objeto; Classe: classe; Local: dentro de métodos; Parâmetro: Na assinatura do método.

Variáveis não podem ter o mesmo nome; não podem começar com números; embora seja permitido, não é uma boa prática começar com "_" e nem "$"; Não é permitido espaços.

variáveis dentro de métodos sempre devem ser inicializadas com um valor inicial.

"final" é uma constante.

Casting (conversões)

Transformar uma determinada variável de um tipo menos específico para um tipo mais específico.

Upcast subir para um tipo com mais capacidade de armazenamento ou apresentação. <- Implícito.
Downcast é rebaixar para uma variável de menor capacidade de armazenamento. <- Explícito.
